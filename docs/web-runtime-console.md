# Web Runtime Console

`/control-plane` is a local example console for the runtime beta. It reads committed example files from the repo during local build.

## Sections

- agent inventory
- tool inventory
- policy profile summary
- review queue
- runtime incident timeline
- policy hit details
- Phase 2 report, compare, Guard decision, and replay examples

## Data Sources

- `docs/examples/runtime/support-agent-profile.json`
- `docs/examples/runtime/review-queue.json`
- `docs/examples/traces/runtime-incident.jsonl`
- `docs/examples/reports/*.json`
- `docs/examples/traces/sample-trace.jsonl`

The page is intentionally local/example based. It does not claim hosted dashboards, accounts, billing, compliance readiness, or production gateway deployment.
